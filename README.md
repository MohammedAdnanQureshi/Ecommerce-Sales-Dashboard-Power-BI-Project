Ecommerce Sales Dashboard | Power BI Project

Excited to share my latest Power BI dashboard where I analyzed ecommerce sales data to uncover key business insights 📊
🔍 Key Insights:
 • Total Sales: 438K+ with 37K profit
 • COD dominates payments (44%), followed by UPI (21%)
 • Clothing leads category sales (63%)
 • Maharashtra contributes the highest revenue among states
 • Profit trends show fluctuations with peaks towards year-end
🛠 What I worked on:
 • Data cleaning & transformation using Power Query
 • Data modeling and KPI creation
 • Interactive dashboards with filters (Quarter & Category)
 • Clear and impactful data storytelling
This project helped me improve my ability to turn raw data into meaningful insights that support decision-making.
